# Gastronomic Festival Website

Responsive website built with HTML, Sass and JavaScript.

## Features

- Responsive layout
- Modular SCSS architecture
- Optimized assets using Gulp
- Deployed with Netlify

## Tech Stack

HTML  
Sass  
JavaScript  
Gulp  

## Live Demo

https://foodies-festival.netlify.app

## Installation

Clone the repository

git clone https://github.com/llopisar/foodies-festival

Install dependencies

npm install

Run development server

npm run dev
